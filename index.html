10. ¿Qué son los métodos / verbos de petición HTTP? Dar ejemplos.

R: Los métodos de petición HTTP son verbos que indican la acción que se desea realizar para un recurso determinado
Ejemplos:
· GET: Solicita una representación de un recurso específico. Las peticiones que usan este método GET sólo deben recuperar datos

· POST: El método POST se utiliza para enviar una entidad a un recurso en específico, causando a menudo un
cambio en el estado o efectos secundarios en el servidor

· HEAD: Este método pide una respuesta similar a la de una petición GET, pero sin el cuerpo de la respuesta.

· PUT: El modo PUT reemplaza todas las representaciones actuales del recurso de destino con la carga útil de la petición.

· DELETE: Este método borra un recurso en específico.
